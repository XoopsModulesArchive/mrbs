Hoe werken thema's in MRBS?

Thema's in MRBS werken volledig onafhankelijk van het gebruikte website thema.
Het bestand mrbs.css.php in /modules/mrbs bevat de gedeelde CSS instellingen voor alle MRBS thema's.
MRBS thema's kan je vinden in  /modules/mrbs/Themes en hebben standaard de extensie dot inc
Twee thema's zijn standaard aanwezig classic126.inc en default.inc

  • Als je een nieuw thema wil maken / gebruiken plaats het in deze folder.
  • Als je een nieuw gemaakt thema wil gebruiken dien je het eerst te activeren door je module te updaten, anders is het niet zichtbaar in de keuzelijst bij instellingen.

Waar kan ik de taalbestanden vinden?

MRBS heeft twee verschillende taalsystemen.

Backend systeem (admin sectie).
Gebruikt het normale xoops taal systeem. Alle bestanden kan je vinden in  /modules/mrbs/language/jouwtaal.

Frontend systeem (werkt onafhankelijk van jouw xoops website instellingen!)
Gebruikt taalbestanden die je kan vinden in /modules/mrbs en deze noemen lang dot taalcode
voorbeeld: nederlands taal bestand noemt lang.nl

MRBS gebruikt een eigen systeem om de frontend taal te kiezen en gebruikt dus NIET het xoops website systeem om dit te doen !!
Hierdoor is de MRBS frontend standaard meertalig.

Ik heb een taalbestand en/of nieuw faq bestand gemaakt maar kan het niet vinden bij de instellingen?

  • Heb je het bestand bewaard in /modules/mrbs?
  • Controleer de naam van je taalbestand -> lang dot taalcode -> voorbeeld lang.nl
  • Controleer de naam van je faqbestand ->  site_faq_taal dot html -> voorbeeld site_faq_nl.html
  • Update je MRBS module -> Xoops Systeem / Modules / Update
  • Ga naar deMRBS instellingen en kies je taal en/of faq bestand
  • Bewaar de instellingen

Waar kan ik de gebruikte tekst voor periodes (periods) en soort (entry type) aanpassen?

De frontend van MRBS gebruikt zijn eigen los van Xoops taalsysteem. Deze bestanden noemen lang dot taalcode.
Voorbeeld: nederlands taal bestand noemt lang.nl

Omdat de frontend van MRBS standaard meertalig is zal je dus alle gewenste taalbestanden moeten aanpassen.
Zoek naar het onderstaande in de taalbestanden rond lijn 258.Deze woorden worden gebruikt voor 'soort' (entry types).

//xoops - blueteen - entry types
$vocab["A"] = "Cat1";
$vocab["B"] = "Cat2";
$vocab["C"] = "Cat3";
$vocab["D"] = "Cat4";

Zoek naar het onderstaande in de taalbestanden rond lijn 285. Deze woorden worden gebruikt voor periodes (periods).

// MRBS assumes that the descriptions are valid HTML and can be output
// directly without any encoding. Please ensure that any special characters
// are encoded, eg '&' to '&amp;', '>' to '&gt;', lower case e acute to
// '&eacute;' or '&#233;', etc.
$vocab["P1"] = "Period&nbsp;1";
$vocab["P2"] = "Period&nbsp;2";
$vocab["P3"] = "Period&nbsp;3";
